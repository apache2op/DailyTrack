#  📈 Daily Track

A simple, minimal, and responsive **habit tracking web app** to help you stay consistent and visualize your daily progress.

---

## ✨ Features

* 📅 **Monthly Habit Tracking**

  * Automatically adjusts days (28 / 30 / 31)
  * Switch between months easily

* 🎯 **Custom Habits**

  * Add habits with custom names
  * Choose your own colors

* 🎨 **Live Color Preview**

  * Instantly see selected color before adding habit

* 📊 **Progress Tracking**

  * Visual progress bar
  * Percentage completion for each habit

* 🌙 **Dark Mode**

  * Clean UI for low-light usage
  * Smart contrast for readability

* 📱 **Responsive Design**

  * Optimized for mobile and desktop
  * Clean, compact layout

* 💾 **Local Storage**

  * Data persists even after refresh

* 📥 **Export Data**

  * Download your habits as a CSV file

---

## 🧠 How It Works

### 1. Add a Habit

* Enter habit name
* Pick a color
* Click **Add**

### 2. Track Progress

* Click on a day box to mark it complete
* Click again to unmark

### 3. Visual Feedback

* Colored boxes represent completed days
* Progress bar shows completion %
* Text color auto-adjusts for readability

### 4. Monthly Tracking

* Select a month from dropdown
* Each month stores data separately

### 5. Export

* Click **Export** to download your data in **CSV** format
